Body

The request body. These are the endpoint parameters.

Typically, this is some request DTO (like for example ConvoCreationRequestDto) that was serialized into JSON and compressed.

If the body is really short and/or represents only a single value, you can also assign the value directly instead of serializing + compressing it.

Signature

The request's signature.

The signed data should be the sum of the three fields (UserId + Auth + Body).

Only ever sign using the user's private message RSA key that the server does not have!
