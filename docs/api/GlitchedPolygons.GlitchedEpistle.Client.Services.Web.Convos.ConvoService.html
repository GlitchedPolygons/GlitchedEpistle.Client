ChangeConvoMetadata(EpistleRequestBody)

Changes a convo's metadata (description, title, etc...).

The user making the request needs to be the Convo's admin (Creator).

If you're assigning a new admin, he needs to be a participant of the Convo, else you'll get a bad request returned from the web api.
