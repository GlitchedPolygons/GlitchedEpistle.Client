StartAutoFetchingMessages(String, String, Int64, Action<IEnumerable<Message>>, Action, Int32)

Starts fetching messages from an Epistle convo continuously.

The retrieved Messages are raw, as they were returned from the backend (thus encrypted using the GlitchedPolygons.GlitchedEpistle.Client.Services.Web.Convos.MessageFetcher.user's private key).

Type Description
System.Threading.CancellationTokenSource

A message fetch routine handle in the form of a System.Threading.CancellationTokenSource, which provides the ability to stop auto-fetching via CancellationTokenSource.Cancel() OR StopAutoFetchingMessages(CancellationTokenSource). DO NOT FORGET to dispose the System.Threading.CancellationTokenSource via either a using block or by calling CancellationTokenSource.Dispose() manually!

StopAutoFetchingMessages(CancellationTokenSource)

Stops continuously fetching a convo's messages via the System.Threading.CancellationTokenSource that was returned by StartAutoFetchingMessages(String, String, Int64, Action<IEnumerable<Message>>, Action, Int32).

Basically does the same as calling the .Cancel() method on the cancellationTokenSource.
