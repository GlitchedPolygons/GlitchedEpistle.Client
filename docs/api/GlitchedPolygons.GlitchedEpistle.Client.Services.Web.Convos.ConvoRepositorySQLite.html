ConvoRepositorySQLite(String)

Creates an instance of the ConvoRepositorySQLite class that will provide functionality for accessing the local Convo storage database using SQLite.

This db only contains the metadata for the convos stored locally on the device.

If the provided connection string points to a file that doesn't exist or to a db that does not contain the convos table, a correct SQLite database + table will be created at that path.
