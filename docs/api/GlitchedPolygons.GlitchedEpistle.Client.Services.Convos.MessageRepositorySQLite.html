MessageRepositorySQLite(String)

Creates an instance of the MessageRepositorySQLite class that will provide functionality for accessing an epistle Message storage database using SQLite.

If the provided connection string points to a file that doesn't exist or a db that does not contain the messages table, a correct SQLite database + table will be created at that path.
